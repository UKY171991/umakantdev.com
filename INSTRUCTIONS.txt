1. DEPLOYMENT & INSTALLATION (Server Setup)
--------------------------------------------------
For fresh installations or after uploading files to a new server:

URL: /install.php
This one-click installer performs:
- Automatic .env creation (if missing)
- Professional cleanup of bootstrap caches
- Generation of Application Encryption Key
- Running Database Migrations
- Linking Storage (Storage Link)
- Final Configuration Caching

--------------------------------------------------
2. MAINTENANCE & UTILITY ROUTES
--------------------------------------------------
The following URLs are available for ongoing management:

A. Clear All Cache
   URL: /clear-cache
   Used for: Fixing "Internal Server Error" or when changes don't appear.

B. Database Migrations
   URL: /run-migrate
   Used for: Updating the database schema manually.

C. Composer Update
   URL: /run-composer
   Used for: Running 'composer update' via browser. (May take a few minutes)
